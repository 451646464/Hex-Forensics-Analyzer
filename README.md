## 🧠 Overview

**Advanced Hex File Extractor & Builder** is a powerful desktop application designed to:

* Extract hidden or embedded files from raw hex data
* Analyze binary structures
* Reconstruct original files with high accuracy

💡 Ideal for:

* Malware Analysis
* Digital Forensics
* Reverse Engineering
* File Recovery

---

## ⚡ Key Features

### 🔍 Smart Extraction Engine

* Detects file boundaries automatically
* Supports multi-file extraction from a single hex stream
* Handles corrupted or mixed data

### 📂 Multi-Format Support

* 🖼️ Images: JPG, PNG, BMP
* ⚙️ Executables: EXE, DLL
* 📄 Documents: PDF, TXT
* 🎵 Media: MP3, MP4, MOV

### 📊 Deep Analysis

* File signatures detection
* Header & trailer inspection
* ASCII visualization
* MD5 hashing

### 🔨 Reconstruction System

* Rebuild original files from extracted hex
* Save raw hex files
* Export JSON reports

### 🖥️ Advanced GUI

* Clean & modern Tkinter interface
* Multi-tab navigation
* Real-time console logs
* Progress tracking

## 🚀 Getting Started

### 📥 Clone the Repository

```bash
https://github.com/451646464/Hex-Forensics-Analyzer.git
cd Hex-Forensics-Analyzer
```

---

### 🧪 Setup Environment

```bash
python -m venv .venv
.venv\Scripts\activate
```

---

### 📦 Install Dependencies

```bash
pip install -r requirements.txt
```

---

### ▶️ Run the Application

```bash
python Hex_Bulding_GUL.py 
```
